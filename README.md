# php-base-image for laravel

PHP base image for Laravel with PHP version 8.2 debian based.
The image contains important packages for laravel.

Packages are included:

* nginx
* cron
* supervisor
* dom
* ctype
* exif
* gd
* intl
* opcache
* pdo
* pdo_mysql
* session
* phar
* xml
* fileinfo
* simplexml
* sockets
* zip
